# Projet QCM Tech

## Description
Une application interactive de questions à choix multiples (QCM) conçue pour tester et améliorer vos connaissances en technologie.

## Fonctionnalités
- 📝 **Questions variées** : Couvrant différents domaines tech
- ⚡ **Interface intuitive** : Facile à utiliser et responsive
- 📊 **Suivi des résultats** : Visualisez votre progression
- 🎯 **Catégories multiples** : Organisés par thèmes
